# Creación de Propuestas con IA para Desarrollo Web - Ejercicio Práctico
## Natalia Rolón Leal

### Prompt inicial
"Soy una desarrolladora de software junior, un cliente mío necesita crear un sitio web. El nombre de la empresa es LauVet, el propósito es generar confianza en la audencia(que en general, van a ser clientes que tengan mascotas), para asegurarles un mejor cuidado y atención veterinario para sus queridas mascotas".

### Prompt completo
"Actúa como un desarrollador web junior y diseñador UX/UI profesional.

Necesito crear un sitio web moderno y profesional para una clínica veterinaria llamada “LauVet”.

Objetivo del sitio:
Transmitir confianza, profesionalismo, cercanía y amor por las mascotas, para que los clientes se sientan seguros dejando el cuidado de sus animales en manos de la clínica.

Público objetivo:
Dueños de mascotas, especialmente familias con perros y gatos, que buscan atención veterinaria confiable, cálida y profesional.

Estilo visual:
- Moderno y minimalista
- Diseño limpio y elegante
- Colores suaves relacionados con salud y mascotas (azul, rosado, blanco y tonos pastel)
- Tipografía sans-serif moderna
- Mucho espacio en blanco
- Diseño emocional y amigable

Secciones del sitio:
1. Hero section con mensaje principal y botón de contacto
2. Sobre nosotros
3. Servicios veterinarios
4. Testimonios de clientes
5. Galería de mascotas
6. Información de contacto
7. Footer profesional

Funcionalidades:
- Diseño totalmente responsive
- Navbar fija
- Scroll suave
- Animaciones suaves al hacer scroll
- Botón flotante de WhatsApp
- Hover effects en tarjetas e imágenes
- Formulario de contacto funcional

Tecnologías:
- HTML5
- CSS3
- JavaScript
- Diseño responsive mobile-first

Genera:
- Código limpio y organizado
- Buenas prácticas
- Comentarios en el código
- Estructura semántica
- Diseño profesional visualmente atractivo

Haz que el sitio se vea como una página moderna real de una clínica veterinaria premium."
